# 家計簿くん AI (LINE Budget Tracker)

LINEから話しかけるだけで、AIが支出を自動解析・記録・グラフ化してくれる自分専用の家計簿システムです。

## 🌟 プロジェクト概要
市販の家計簿アプリの「手入力の面倒さ」を解消するために開発しました。LINEのチャット画面に「コーヒー 150円」のように送るだけで、AIが項目、金額、カテゴリーを抽出してGoogleスプレッドシートへ記録します。

## 🚀 主な機能
- **AI支出記録**: 自然言語（「昨日ランチで1200円使った」等）から、日付・項目・金額を自動抽出。
- **カテゴリー自動推論**: 項目名に基づき、「食費」「日用品」「交際費」等のカテゴリーをAIが自動分類。
- **動的なグラフ生成**: 「今月のグラフ」「先月の内訳」等の指示に応じて、リアルタイムで円グラフを生成してLINEに返信。
- **Googleスプレッドシート連携**: 記録データはクラウド上に保存されるため、PCからの閲覧や分析も容易。

## 🛠 使用技術（Tech Stack）
- **Language**: Python 3.10+
- **Infrastructure**: Render (Server), GitHub
- **APIs**:
  - OpenAI API (GPT-4o-mini)
  - LINE Messaging API
  - Google Sheets API / Google Drive API
- **Main Libraries**:
  - `Flask`: Webサーバー
  - `Matplotlib` / `japanize-matplotlib`: グラフ描画
  - `gspread`: スプレッドシート操作
  - `line-bot-sdk`: LINEとの連携

## 🏗 システム構成図


1. **User**: LINEでメッセージを送信。
2. **LINE Platform**: Webhookを介してRender上のFlaskサーバーに通知。
3. **Flask Server (Render)**:
   - OpenAI APIを呼び出し、自然言語を解析。
   - Google Sheets APIを介してデータを書き込み。
   - 必要に応じてMatplotlibでグラフ画像を生成。
4. **Result**: 解析結果のテキストやグラフ画像をユーザーへ返信。

## ✨ こだわったポイント
- **UXの最適化（高速化）**: LINEの5秒タイムアウトを回避するため、意図解析とデータ抽出のプロンプトを統合。1回のAPIリクエストで完結させることで、ストレスのない応答速度を実現しました。
- **キャッシュ対策**: 生成したグラフ画像URLの末尾にタイムスタンプを付与することで、LINEアプリ内での画像キャッシュによる「古い情報の表示」を防止しました。
- **データ堅牢性**: AIが「項目：飲み物」のように余計なラベルを返した場合でも、正規化して保存するクリーニング機能を実装しました。
